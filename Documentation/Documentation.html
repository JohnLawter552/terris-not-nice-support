A) HTML Structure - top

This template is a Responsive layout, all contents are within section tag The general template structure is the same throughout the template. Here is the general structure.

If you would like to change the the purple theme color, change the the following in style.css in css folder

	.theme-color{
    	color:#your-color;
}
	
 Also replace this code "color:rgb(152,120,199);" wherever you find it in from style.css with your desired template color

If you would like to change font of the template, you would do the following. Don't forget to add !important to your code.

	body{
    font-family: "Your font name" ,font type !important;
}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.
